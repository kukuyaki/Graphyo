## How It Works

### Code Input (Left Sidebar):

Top (Green): Enter your Graph Generator code.

Middle (Blue): Enter your Pathfinding/Graph algorithm code.

Note: Both inputs should return a JSON object containing the graph and step-by-step execution data. You can use our templates or your own implementation.

### Visualization (Right Panel):

The application automatically renders the JSON output into an interactive graph.

### Controls:

Click the Green button to execute the generator.

Click the Blue button to run your algorithm.

Use the small blue button and the slider to adjust the animation speed and step through the process.
